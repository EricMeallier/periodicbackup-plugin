PeriodicBackup-plugin

by dienomight & emanuelez

Jenkins plugin to backup periodically

The idea of this plugin was to create a backup plugin that could be easily extended with new functionality without the need to hack around with the code.

Dependencies

  • junit
  • plexus-archiver
  • guava
